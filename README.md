# The Last Train Home - Interactive Flipbook Website

This project is a responsive HTML, CSS, and JavaScript story website. It displays the original short story **The Last Train Home** inside an animated flipbook interface.

## Project structure

```text
the-last-train-book-site/
├── assets/
│   └── cover.svg
├── index.html
├── style.css
├── script.js
└── README.md
```

## Features

- Fully responsive layout for desktop, tablet, and mobile screens.
- Custom cover artwork matching the story atmosphere.
- Animated page-turning effect with smoother flip motion.
- Keyboard navigation with left and right arrows.
- Mobile swipe navigation.
- Progress bar and page counter.
- Instagram creator link on the final page.
- Copyright and design credit on the final page.
- Smooth glass-style hero panel.
- Separate HTML, CSS, and JavaScript files.
- English-only visible website content.

## How to run

1. Extract the ZIP file.
2. Open the folder named `the-last-train-book-site`.
3. Double-click `index.html` to open it in a browser.

No installation is required.

## Editing the story

To change the story text, open `index.html` and edit the text inside each `<article class="page">` block.

## Editing the design

To change colors, layout, animation speed, or spacing, open `style.css`.

Main color variables are located at the top of `style.css` inside the `:root` section.

## Editing behavior

To change page navigation logic, animation timing, keyboard controls, or swipe behavior, open `script.js`.
